# Setup onboarding scope

This launch-hardening pass adds a guided setup question pattern across the setup flow.

Covered areas:

- Rate cards
- Accessorials
- Drayage zones
- Brand page
- Assistant setup
- Public link / embed

Each area should use:

- clickable preset answers
- custom answer option
- short plain-language question copy
- saved local guidance so operators can return to the setup step

Assistant setup goals:

- Ask which services the assistant should discuss.
- Ask what the assistant should avoid promising.
- Ask when the assistant should route to the carrier team.
- Ask what tone the assistant should use.
- Allow answers to be applied into the behaviour prompt.
